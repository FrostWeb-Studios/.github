# FrostWeb Studios

Org profile, reusable workflows, workflow templates, and community health files (SECURITY.md, CONTRIBUTING.md, CoC, issue/PR templates).

## Profile

The public-facing profile README lives at [`profile/README.md`](profile/README.md).

## Reusable workflows

All under [`.github/workflows/`](.github/workflows/). Call from any repo via `uses: FrostWeb-Studios/.github/.github/workflows/<name>.yml@master`. See [Workflows.md in .github-private](https://github.com/FrostWeb-Studios/.github-private/blob/master/InternalDocs/Workflows.md) for the full map.

## Workflow templates

Under [`workflow-templates/`](workflow-templates/) — selectable from the Actions UI when adding workflows to any repo.
